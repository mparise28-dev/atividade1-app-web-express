Atividade 1: App Web com Express

Disciplina: Desenvolvimento Web I
Professor: Prof. Arley
Aluno: Marcello Parise

Objetivo

Desenvolver uma aplicação web utilizando Node.js e Express para servir páginas HTML estáticas.

Tecnologias Utilizadas

HTML

CSS (Flexbox)

Node.js

Express

Estrutura do Projeto
app
├ server.js
├ package.json
├ .env
└ public
├ assets
│ ├ css
│ └ img
└ pages
├ index.html
├ login.html
├ cadastro.html
└ 404.html
Rotas da Aplicação

/ → página inicial

/login → página de login

/cadastro → página de cadastro

rotas inexistentes → página 404

Deploy

A aplicação foi publicada na plataforma Render, permitindo acesso através de um link público.

Conclusão

A atividade permitiu compreender a criação de um servidor web com Node.js e Express, a organização de um projeto web e o processo básico de deploy de uma aplicação
